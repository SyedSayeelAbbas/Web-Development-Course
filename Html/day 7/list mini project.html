HTML Lists - Complete Guide

Topic 1: Introduction to HTML Lists

HTML lists allow web developers to group a set of related items in lists.

There are three types of HTML lists:

  1. Unordered List <ul> - items marked with bullets
  2. Ordered List <ol> - items marked with numbers
  3. Description List <dl> - list of terms with descriptions

HTML List Tags

<ul>
Defines an unordered list
<ol>
Defines an ordered list
<li>
Defines a list item
<dl>
Defines a description list
<dt>
Defines a term in a description list
<dd>
Describes the term in a description list

Topic 2: Unordered HTML Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default.

Basic Unordered List

  • Coffee
  • Tea
  • Milk

Unordered List - Bullet Styles (list-style-type)

1. Disc (default)

  • Coffee
  • Tea
  • Milk

2. Circle

  • Coffee
  • Tea
  • Milk

3. Square

  • Coffee
  • Tea
  • Milk

Nested Unordered Lists

Lists can be nested (list inside list):

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Deeply Nested List (3 levels)

  • Beverages
    • Hot Drinks
      • Coffee
      • Tea
      • Hot Chocolate
    • Cold Drinks
      • Juice
      • Soda
      • Water
  • Food
    • Breakfast
    • Lunch
    • Dinner

Chapter Summary - Unordered Lists

  • Use the HTML <ul> element to define an unordered list
  • Use the CSS list-style-type property to define the list item marker
  • Use the HTML <li> element to define a list item
  • Lists can be nested
  • List items can contain other HTML elements

Topic 3: Ordered HTML Lists

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The list items will be marked with numbers by default.

Basic Ordered List

  1. Coffee
  2. Tea
  3. Milk

The type Attribute - All Marker Types

type="1" - Numbers (default)

  1. Coffee
  2. Tea
  3. Milk

type="A" - Uppercase Letters

  1. Coffee
  2. Tea
  3. Milk

type="a" - Lowercase Letters

  1. Coffee
  2. Tea
  3. Milk

type="I" - Uppercase Roman Numerals

  1. Coffee
  2. Tea
  3. Milk

type="i" - Lowercase Roman Numerals

  1. Coffee
  2. Tea
  3. Milk

The start Attribute - Control List Counting

Use the start attribute to begin counting from a specified number:

Start from 5

  1. Coffee
  2. Tea
  3. Milk

Start from 50

  1. Coffee
  2. Tea
  3. Milk

Uppercase letters starting from E (start="5")

  1. Coffee
  2. Tea
  3. Milk

The reversed Attribute

The reversed attribute makes the list count downward:

  1. Gold Medal
  2. Silver Medal
  3. Bronze Medal

The value Attribute on <li>

Use value on individual list items to set a custom number:

  1. Ten points
  2. Twenty points
  3. Thirty points

Nested Ordered Lists

Ordered lists can also be nested:

  1. Coffee
  2. Tea
    1. Black tea
    2. Green tea
  3. Milk

Mixed Nested Lists (Ordered inside Unordered)

  • Fruits
    1. Apple
    2. Banana
    3. Mango
  • Vegetables
    1. Carrot
    2. Spinach
    3. Potato

Chapter Summary - Ordered Lists

  • Use the HTML <ol> element to define an ordered list
  • Use the HTML type attribute to define the numbering type
  • Use the HTML start attribute to begin from a specific number
  • Use the HTML reversed attribute to count backwards
  • Use the HTML value attribute on <li> to set a custom number
  • Use the HTML <li> element to define a list item
  • Lists can be nested

Topic 4: HTML Description Lists

HTML also supports description lists.

A description list is a list of terms, with a description of each term.

The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term.

Basic Description List

Coffee
- black hot drink
Milk
- white cold drink

Description List - Web Technologies Glossary

HTML
HyperText Markup Language - the standard language for creating web pages
CSS
Cascading Style Sheets - used to describe the presentation of HTML documents
JavaScript
A scripting language that enables interactive and dynamic content on web pages
HTTP
HyperText Transfer Protocol - the foundation of data communication on the web
URL
Uniform Resource Locator - the address used to access a resource on the internet

One Term with Multiple Descriptions

Coffee
- A brewed drink prepared from roasted coffee beans
- One of the most popular drinks in the world
- Contains caffeine which helps alertness

Multiple Terms with One Description

Tea
Chai
- A hot beverage made by brewing tea leaves in water

Description List - HTML Tags Reference

<ul>
Defines an unordered list
<ol>
Defines an ordered list
<li>
Defines a list item (used inside ul and ol)
<dl>
Defines a description list
<dt>
Defines a term in a description list
<dd>
Describes the term in a description list

Chapter Summary - Description Lists

  • Use the HTML <dl> element to define a description list
  • Use the HTML <dt> element to define the description term
  • Use the HTML <dd> element to describe the term
  • One <dt> can have multiple <dd> descriptions
  • Multiple <dt> terms can share one <dd> description
